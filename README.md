<h2 id="usage" > 👷 Usage </h2>

Required! Install git, node and yarn (or npm).

```bash
# Clone Repository
$ git clone https://github.com/RodrigoSaantos/permaneo.git

# Go to folder classroom
$ cd permaneo/

# Install Dependencies
$ yarn
# or
$ npm install

# Run Application
$ yarn dev
# or
$ npm run dev

# Access localhost
http://localhost:3000

# Run backend
$ npx json-server -w db.json -p 3333

# Access localhost
http://localhost:3333/
```
